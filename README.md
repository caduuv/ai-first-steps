# 🧬 AI First Steps — Cervical Cytology Conditional Generation Course

A progressive, hands-on Python course that takes you from mathematical foundations to building conditional generative AI models for medical imaging. The final project focuses on synthesizing cervical cytology images conditioned on medical parameters.

---

## 📋 Prerequisites

- **Python 3.10+**
- Basic programming experience (variables, loops, functions)
- Curiosity about AI and medical imaging

No prior machine learning experience required — we start from scratch!

---

## 🗂️ Course Structure

| Module | Topic | Key Skills |
|--------|-------|------------|
| **01** | Mathematical & Statistical Foundations | Linear algebra, probability, gradients, optimization |
| **02** | Python for Machine Learning | NumPy, Pandas, Matplotlib, data pipelines |
| **03** | Deep Learning | Perceptrons, backpropagation, PyTorch |
| **04** | Computer Vision | CNNs, image processing, data augmentation |
| **05** | Generative Models | Autoencoders, VAEs, GANs |
| **06** | Conditional Generation | cGAN, cVAE, diffusion model concepts |
| **07** | Final Project | Cervical cytology image synthesis (documentation & scaffolding) |

Each module contains:
- **README.md** — Theory, concepts, and learning objectives
- **Numbered scripts** — Progressive, well-commented implementations
- **exercises.md** — Practice problems to cement your understanding

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone <repo-url>
cd ai-first-steps
```

### 2. Set Up the Environment

**Windows (PowerShell):**
```powershell
.\setup_env.ps1
```

**Linux / macOS:**
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 3. Start Learning
Open `module_01_fundamentals/README.md` and follow along!

## 💡 How to Use the Scripts

All `.py` files use **`# %%` cell markers** for interactive execution. This means you can:

1. **Run them as regular scripts:** `python 01_linear_algebra.py`
2. **Run interactively in VS Code:** Open the file, and VS Code will recognize `# %%` cells — click "Run Cell" above each section
3. **Run interactively in PyCharm:** Use the "Scientific Mode" to execute cells

This gives you notebook-like interactivity while keeping files version-control friendly.

## ⚠️ Ethics & Responsible AI

This course covers the generation of **synthetic medical images**. Module 7 includes a dedicated document on ethical considerations, including:

- Risks of synthetic data in clinical settings
- Importance of expert validation
- Bias in training data
- Regulatory considerations

**Synthetic images generated by this project are NOT suitable for clinical use.**
