## 📖 Theory Overview

### The Data Science Toolkit

Before building models, you need to:

1. **Load** data from various formats (CSV, JSON, databases)
2. **Explore** it to find patterns, distributions, and anomalies
3. **Clean** it by handling missing values, outliers, and errors
4. **Transform** it into a format suitable for ML (normalization, encoding)
5. **Split** it into training, validation, and test sets

These steps are often 80% of the work in a real ML project!

---

### NumPy: The Foundation

NumPy is the backbone of scientific Python. Key concepts:

- **ndarray**: N-dimensional arrays with fixed types
- **Vectorization**: Avoid Python loops by operating on entire arrays
- **Broadcasting**: Automatic shape alignment for operations
- **Random**: Reproducible random number generation

---

### Pandas: Data Manipulation

Pandas provides the DataFrame — a 2D labeled data structure:

- **Loading**: Read CSVs, Excel, SQL databases
- **Selection**: Filter rows/columns with boolean indexing
- **GroupBy**: Split-apply-combine for aggregations
- **Merging**: Join multiple DataFrames like SQL

---

### Visualization: Telling the Story

Matplotlib and Seaborn help you understand and communicate data:

- **Distributions**: Histograms, KDE plots, box plots
- **Relationships**: Scatter plots, pair plots, heatmaps
- **Trends**: Line plots, time series
- **Model diagnostics**: Residual plots, ROC curves
