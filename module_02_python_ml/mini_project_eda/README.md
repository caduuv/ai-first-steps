# Mini Project — Exploratory Data Analysis

## 🎯 Objective

Perform a complete EDA on the Breast Cancer Wisconsin dataset.
This mini-project ties together everything from Module 2.

## Instructions

Run `eda_analysis.py` to see a full EDA workflow, then modify it to explore:
1. Which features best separate malignant from benign tumors?
2. Are any features highly correlated (redundant)?
3. What does the distribution of each feature look like?

## What You'll Practice
- Data loading and inspection
- Statistical summaries
- Multiple visualization types
- Feature selection intuition
